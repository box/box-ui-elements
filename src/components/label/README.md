### Examples

**Basic**

```
<Label text='Input Label' tooltip='This is an input label.'>
    <input type="text" />
</Label>
```

**With Optional Text**

```
<Label text='Input Label' showOptionalText>
    <input type="text" />
</Label>
```

**With Info Tooltip**

```
<Label text='Input Label' infoTooltip="I stand above this icon">
    <input type="text" />
</Label>
```
